手机端调试

Posted on 2018-10-02 | In JavaScript

0.移动端调试

  • 移动端调试时
  • 由于手机浏览器没有控制台
  • 所以在发生报错的时候（尤其是 JS 报错）
  • 难以调试

1.alert

  • 即便移动端没有控制台
  • 那么就用 alert 将错误通过 alert 对话框的形式发出来

2.VConsole

  • 非常棒的库
  • 可以在移动端模拟控制台
  • 腾讯做的

npm install vconsole
引入dist/vconsole.min.js到项目
