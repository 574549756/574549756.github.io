配置vim

Posted on 2018-10-25

0. 首先需要检查 vim 是否支持 clipboard

终端输入
vim --version | grep "clipboard"
检查 vim 版本是否支持 clipboard
如果显示 -clipboard 则说明不支持

I. 安装图形化界面的 vim

sudo apt get install vim-gnome
安装完成后再次运行
vim --version | grep "clipboard"
发现之前的减号变成了加号+

这时 vim 已经支持 clipboard 了

1. 导入 vimrc 配置 vim

已上传 github
快捷键space + y即可复制到系统剪贴板
