1.AE 箭头如何制作

I.箭头的柄

  • 首先新建一个形状图层
  • 钢笔工具绘制路径(末端开始,不要填充,要路径描边,设置宽度)
  • 点击 add 右边的小箭头
  • 添加路径修剪(我用的英文版不知道你用的是怎么叫的)
  • 点击内容左边的三角箭头
  • 看展开里面有一个 start
  • 在动画开始的时候把 0% 改成 100%(全部裁剪)
  • 点击一下左边的小闹钟
  • 拖动时间轴到动画结束的时候(路径绘制完成的时间点)
  • 在动画开始的时候把 100% 改成 0% (不裁剪)
  • ok 这个箭头的箭的补间动画就完成了

tip: 如果需要 dash 线的话在图层控制面板-内容-描边-虚线-点击加号

II.箭头的箭头

  • 首先我们需要一个等腰三角形
  • 多边形工具绘制
  • 展开内容
  • 找到图形路径,展开
  • 找到点菜单,右侧显示 5
  • 这是控制点数量,将其改成 3
  • 现在我们拥有了一个等边三角形
  • 将旋转属性改为 0×0
  • 拖动底角的控制点
  • 这样我们就拥有一个等腰三角形了

i.添加箭头动画

  • 在动画开始的时候把 0% 改成 100%(全部裁剪)
  • 点击一下左边的小闹钟
  • 拖动时间轴到动画结束的时候(路径绘制完成的时间点)
  • 在动画开始的时候把 100% 改成 0% (不裁剪)

III.箭头的撤出

  • 做完之前的操作我们了解到通过关键帧控制补间动画很像 flash
  • 那么箭头的撤出也相同
  • 控制撤出时间,在需要撤出的时候做进入时相反的操作就好了

IV.文字的进入

  • 我们了解了补间动画如何完成
  • 那么透明度也可以作为补间(任何你看到闹钟的都可以设置关键帧)
  • 所以可以用同样的方法完成补间动画

2.画笔随路径移动

  • 素材拖入时间轴
  • 移动素材的位置,创建关键帧(开始和结束)
  • 每次改变运动方向需要创建新关键帧
  • 移动路径会用虚线显示出来
  • 拖动锚点改变移动路径,形成平滑的贝兹曲线
  • 图层-变换-自动定向
  • 如果需要素材随路径移动则开启沿路径定向
  • 自定义移动速度需要点击图形曲线(在图层面板的右上角)
  • 更改动画速率
  • x 为时间轴 y 为速度
  • 按需要调整
  • 位置路径可以拷贝粘贴到其他素材的路径位置(如果需要两个素材同步的话)

tip:全选所有关键帧按住 alt 键拖移可以均匀调整动画时间
